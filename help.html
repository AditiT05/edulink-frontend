Getting Started

What is EduLink?

EduLink is a real-time classroom interaction system designed to improve communication between teachers and students.

How to use EduLink?

Navigate using the sidebar to access Dashboard, Calendar, Announcements, Settings and Help.

For Teachers

Create Classes

Teachers can create and manage classes from the Dashboard section.

Post Announcements

Announcements can be created with messages and attachments to communicate important updates to students.

Schedule Events

The Calendar allows scheduling events across multiple dates and years.

For Students

View Classes

Students can see enrolled classes on their dashboard.

Read Announcements

Students receive announcements posted by teachers in real time.

Frequently Asked Questions

Can students create announcements?

No, only teachers can create announcements.

Is EduLink secure?

EduLink follows secure development practices. Future versions will include enhanced authentication and data protection.
